大赛精神

  • 激发创意，实际创新，一展成都程序员风采 紧跟时代
  • 力争一流，彰显成都IT新一线城市风貌
  • Code For City，用实际的编程行动改变天府生活，让成都更加美好
  • 倡导人人皆可编程理念，传达编程的美妙
  • 为成都高新科技人才提供一个展示擂台，扩展各大高新企业的人才储备

大赛目的

  • 企业高科技产品展示、程序员的狂欢节、提高成都软件行业社区影响力、鼓励创新创意与实践
  • 打通技术人员与企业之间的沟通通道，促进企业相互交流
  • 创造有潜力的IDEA与产品，吸引投资方投资有实力的团队
  • 提高团队协作能力，加强高新区与软件园对人才的吸引力、积累企业之间互相协助沟通的经验
